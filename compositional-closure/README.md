# 元函数组合近似封闭性（CAC）

> 本目录记录关于大模型涌现能力的形式理论：模型学到的函数集合 $F$ 通过组合，能到达任何可由原始逻辑原语 $R$ 组合到达的计算终点。

## 核心文档

- [`theory.md`](./theory.md) — 完整形式理论（设定、CAC 假说、误差分析、涌现机制、与行为吸引子的关系）

## 与其他模块的关系

- [`behavioral-attractors/`](../behavioral-attractors/) — CAC 为行为吸引子的**存在性**提供了基础（每个吸引子是稳定的 $F$-链路构型）；行为吸引子理论描述这些构型的**拓扑结构**
