# 大语言模型中的行为吸引子

> **研究方向**：大语言模型参数空间中是否存在有限数量的稳定行为吸引子盆地？它们能否被主动设计、测量，并用于解释模型的涌现式多人格行为？

## 起源

本研究方向起源于对一种新型训练范式的思考——**伴生模型协同训练**——它介于稠密训练与稀疏（MoE）训练之间。最初的工程直觉逐步演化为一个更基础的理论问题：LLM 行为空间的内禀几何结构是什么？

## 核心命题（置信度递减）

| # | 命题 | 状态 |
|---|---|---|
| 1 | 单一模型的参数空间中可以包含多个结构性不同的行为模式 | 假设，部分得到越狱研究文献支持 |
| 2 | 伴生协同训练能主动设计这些模式，且其分离程度深于事后 LoRA | 新颖假设，可实验验证 |
| 3 | Prompt 引发的行为偏移与 adapter 激活的设计模式共享几何签名 | 新颖假设，可实验验证 |
| 4 | 这意味着稳定吸引子解的存在性独立于激活路径 | 由命题 3 推导的理论推断 |
| 5 | 稳定吸引子的数量有限且可枚举 | 推测性，长期目标 |
