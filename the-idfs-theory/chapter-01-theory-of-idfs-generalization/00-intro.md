# 第一章 IDFS 的泛化理论基础

## 引言：从连续统计到离散拓扑

在经典的统计学习理论（如 PAC 学习、VC 维理论）中，模型泛化被理解为**在连续的高维流形上对未知数据点的概率插值**。这种范式假设了数据分布的平滑性，并将模型视为一个参数化的统一连续函数。然而，当一个系统由**有限个离散组件经路由选择后多步复合**而成——无论它是大语言模型（LLM）中的 token 级逐步推理、多智能体协作中的任务链路传递，还是任何由"选择 → 执行 → 再选择"构成的迭代决策系统——连续插值的视角开始失效。系统不再仅仅是"拟合曲线"，而是在执行序列化的、离散的**逻辑跳转与状态变异**。

本理论提出一种全新的数学刻画：**输入驱动函数系统（Input-Driven Function System, IDFS）**。

IDFS 截断了所有的统计概率假设与神经网络内部的生化隐喻，将其抽象为一个纯粹的**拓扑动力系统**。在这个系统中，泛化不再是"概率分布在流形上的收敛"，而是**由一个离散的控制中枢（路由 $\sigma$）在有限的函数库（基底 $F$）中进行选择，通过多步复合操作构造出逼近目标算子的长链流形的最坏情况拓扑边界**。

本章建立 IDFS 的基本数学框架。我们对被拟合的空间 $\mathcal{X}$ **不引入任何假设**——不假设它是 $\mathbb{R}^d$、不假设它有向量空间结构、不假设它具有可微性或紧致性、不依赖任何优化算法（如梯度下降）的收敛性。唯一的起点是 $(\mathcal{X}, d)$ 构成一个**度量空间**这一最低限度的数学结构。在这个纯粹的拓扑骨架上，我们考察由微观离散采样强行黏合而成的宏观组合体，在执行多步推理时究竟受制于怎样的绝对物理极限。

### 章节路线图

- **§1 基本定义**：建立度量空间 $(\mathcal{X}, d)$、变换空间 $\Omega$、采样集 $\mathcal{S}$、IDFS $(F, \sigma)$、Lipschitz 约束等核心数学对象。
- **§2 IDFS 的结构性质**：从拓扑容量极限、路由分辨率、长链扩张/收缩等角度，推导 IDFS 的内在结构约束。
- **§3 上界定理（CAC）**：建立复合逼近链（Compositional Approximation Chain）定理，给出多步链误差的精确上界。
- **§4 下界定理（CAB）**：建立复合逼近势垒（Compositional Approximation Barrier）定理，给出链误差的不可突破下界。
- **§5 结构特异性**：揭示 IDFS 单映射共享架构产生的一系列反直觉现象——方向不对称性、采样域劫持、采样集扩增的非单调性。
- **§6 受限 IDFS**：在 CAC 与 CAB 联合约束下，推导收缩走廊、不可能三元困境、以及三种工程构型的数学必然性。
- **§7 Type B 系统理论**：深入分析唯一可行构型（Type B），建立分段复合的误差隔绝定理与容量-精度-深度的统一缩放律。
- **§8 本章总结**：主定理链的浓缩叙述、核心不等式汇总、三种构型的终极画像，以及向第二章（拟合形态学）的过渡。
