## 本章总结

### 主定理链

本章从度量空间 $(\mathcal{X}, d)$ 这一最低限度的数学结构出发，建立了输入驱动函数系统（IDFS）的完整泛化理论。全章的逻辑弧线形成一条不可逆的推导链：

**基础层**（§1–§2）定义了 IDFS $(F, \sigma)$ 的核心数学对象——有限基函数库 $F$、路由映射 $\sigma$、采样集 $\mathcal{S}$、路径局部 Lipschitz 乘积 $\Theta_{j,l}$、路径 Lipschitz 跨度 $\kappa_\Phi$——并推导了系统的内在结构约束：路由容量的组合上界（命题 2.1）、$f$-链正交性理论（命题 2.11–2.13，建立了结构正交→变分正交的蕴含关系，以及正交性对 CAC 五项中 $\Delta^{err}$、$\Delta^{sam}$ 的降阶/消去效应）、长链扩张/收缩动力学（命题 2.6–2.9）、路由分辨率极限（命题 2.7）、等效张力跨度 $\gamma_R$ 对基函数库跨度的传导（命题 2.11）。

**边界层**（§3–§4）建立了互补的端到端误差定理。**CAC 定理**（§3）给出**五项精细上界**——将误差源分解为逐步拟合残差 $\varepsilon_{i_j}$、采样域偏离 $\delta_j$、目标域外变分 $\rho_j$、路由失准惩罚 $\Delta^{err}_j$、采样域路由失配 $\Delta^{sam}_j$ 五个独立可控的分量，不假设目标规则的 Lipschitz 性。**CAB 定理**（§4.1）给出变分下界 $\varepsilon^*_y \ge |\Delta - \varepsilon_x| - \Omega_l(\delta)$；**CPI 定理**（§4.2）给出路由容量的覆盖数必要条件 $|\mathrm{Im}(\sigma)| \ge \mathcal{N}(A, 2\varepsilon/k) / \mathcal{N}(A, \varepsilon/L)$；**DFG 定理**（§4.3）证明不完美拟合集 $U_\tau$ 具有不可消除的正测度下界。三条下界从变分、容量、测度三个层面封锁系统的泛化挣扎空间。

**约束层**（§5–§7）揭示了 IDFS 单映射共享架构的结构性代价。§5 证明了三类反直觉现象——方向不对称性（命题 5.1–5.2）、采样域劫持及变分正交侧向逃逸（命题 5.3，推论 5.4–5.5）、采样扩增的非单调性与强制退化（命题 5.6，推论 5.7）。§7 将 CAC 与 CAB 联合施加于同一系统：§7.1 推导 Lipschitz 乘积双界夹击（命题 6.1）→ 收缩走廊（推论 6.2–6.4）→ 完美即平庸定理（定理 6.11），其中对偶陷阱（推论 6.8）经全局预算闭合（推论 6.9）与 Pareto 前沿（推论 6.10）获得五项精细化加强。§7.2 推导泛化-深度-控制不可能三元困境（命题 6.12）与路由碎裂的上下界对偶博弈（命题 6.13），并精确刻画三种构型：Type A 查表退化与采样诅咒（命题 6.14–6.15）、Type B 走廊解除与代偿定律（命题 6.16–6.17）、Type C 双峰不可消除性与灾难跳变（命题 6.18–6.20）。构型优选推论（推论 6.21–6.22）从误差全域有界性、采样非指数爆炸、抗劫持鲁棒性三条独立维度论证了 **Type B 是唯一可行构型**。

**特化层**（§8）将全部理论精确特化到 Type B 约束下。命题 7.1 证明长链平庸性的四重缺陷全部消解；命题 7.2 证明 CAC 域外项（$\rho_j$、$\Delta_{\sigma,j}$）的短链免疫与 $\Delta_\sigma$ 从零和博弈到正和博弈的解放；命题 7.3 建立 $M$-精度走廊。**误差隔绝定理**（定理 7.4）证明了在码本间距条件 $\tau < \Delta_\mathcal{C}/2$ 下，离散重置将跨段乘性误差累积**精确归零**，使系统获得等效无限深度（推论 7.5）。码本 $\mathcal{C}$ 被赋予三重正当性：代数截断（定理 7.4）、信道编码（推论 7.5）、反劫持锚点（§7.2.4 对接）。定理 7.6 给出一般段间误差递推，推论 7.7 刻画三体制（收敛/临界/发散）。§8.3 将代偿约束、隔绝约束、覆盖约束统一为闭合缩放律（命题 7.8–推论 7.10）。

### 核心不等式汇总

| 不等式 | 名称 | 来源 | 含义 |
|--------|------|------|------|
| $\varepsilon^*_q \le \sum_j (\varepsilon_{i_j}+\rho_j+\Delta^{err}_j)\Theta_{j+1,l} + \sum_j (\delta_j+\Delta^{sam}_j)\Theta_{j,l}$ | CAC 五项精细上界 | §3 定理 | 五类误差源各自独立量化，经 Lipschitz 乘积逐步放大 |
| $\varepsilon^*_y \ge \lvert\Delta - \varepsilon_x\rvert - \Omega_l(\delta)$ | CAB 变分下界 | §4.1 定理 | 目标变分强制系统承受不可压缩残差 |
| $\lvert\mathrm{Im}(\sigma)\rvert \ge \mathcal{N}(A, 2\varepsilon/k) / \mathcal{N}(A, \varepsilon/L)$ | CPI 容量-精度不等式 | §4.2 定理 | 判别性目标对路由多样性的刚性需求 |
| $\mu(U_\tau) \ge \mu(\mathcal{X}_r)\bigl(1 - \lvert\mathrm{Im}(\sigma)\rvert \cdot e^{C_\varepsilon} / e^{I_\varepsilon}\bigr)^{1/\beta}$ | DFG 正测度下界 | §4.3 引理 | 不完美拟合集不可消除 |
| $\zeta^{-1/l} \le \bar{L} \le \frac{\bar{L}}{L_{max}}(1 - 1/l^*_0)$ | 收缩走廊 | 推论 6.3 | 长链 $\bar{L}$ 被双向夹击 |
| $l_{max} = \ln\zeta / (\ln(L_{max}/\bar{L}) + 1/l^*_0)$ | 最大可行链深 | 推论 6.4 | 泛化性碾压安全链深 |
| $\Delta_{max}\Lambda_l \le B_\sigma = \tau - (\varepsilon_{max}+\rho_{max})\Lambda_l - \delta_{max}\Gamma_l$ | 路由碎裂博弈 | 命题 6.13 | $\Delta_\sigma$ 的 CAC 容限被 CAB 对偶消耗 |
| $M \ge \exp((I_\varepsilon - C_\varepsilon)/l^*_0)$ | 代偿定律 | 推论 6.17 | 短链以 $M$ 指数膨胀换深度 |
| $\tau \lt \Delta_\mathcal{C}/2$ | 误差隔绝条件 | 定理 7.4 | 段间误差精确归零 |
| $\varepsilon_k \le L^{l_0} \cdot \varepsilon_{k-1} + \delta_\pi + \tau$ | 段间递推 | 定理 7.6 | 一般段间误差传播 |

### 三种构型的终极画像

| 维度 | Type A：死记硬背 | Type B：短链高保真 | Type C：容忍崩溃 |
|------|-----------------|-------------------|-----------------| 
| **放弃** | 泛化性（$\gamma_R \approx 1$） | 链深（$l \le l^*_0$） | 全路径控制 |
| **走廊** | 极宽（跌落不动点） | **不适用**（前提不触发） | 极窄（甚至为空） |
| **路由** | 宏观完全简并（$\kappa_\Phi \approx 1$） | 极度细化与差异化（$\kappa_\Phi \gg 1$） | 安全域内差异化 |
| **$\Delta_\sigma$** | $\approx 0$：精度优势 + 泛化无能对偶 | 正和博弈工具（CAC 代价 $O(l^*_0)$ 截断） | 零和博弈筹码 |
| **$\rho_j$ 漂移** | 不相关（无域外推理） | 短链截断，$\rho$ 有界 | 长链累积，$\rho$ 随 $\delta$ 指数发散 |
| **正交降阶** | 不可用（$\kappa_\Phi \approx 1$ 致维度坍缩） | 充分可用（$\Delta^{err} \sim O(\sqrt{l})$） | 不可用（临界单次跳变） |
| **采样** | $(D\lvert\lambda_r-\bar{L}^l\rvert/\varepsilon)^d$ 全域穷举 | 自适应异质网格 | 边界易碎覆盖 |
| **长链策略** | 内部迭代（受走廊保护） | **分段复合 + 离散重置** | 内部迭代（受走廊挤压） |
| **误差隔绝** | 不需要（走廊内自稳） | **$\tau \lt \Delta_\mathcal{C}/2$ 归零** | 不适用（失控域发散） |
| **抗劫持** | 自毒化（命题 5.6） | 短链截断 + 码本锚点 | 劫持驱动灾难跳变 |
| **最糟失败** | 维度诅咒级过拟合 | 单次映射计算深度瓶颈 | $\delta_{adv}$ 极小致幻崩溃 |
| **可行性** | ❌ 采样指数爆炸 | ✅ 唯一可行构型 | ❌ 正测度失控域 |

### 本章的绝对性

本章的全部定理均建立在度量空间 $(\mathcal{X}, d)$ 这一最低限度结构之上。不假设 $\mathcal{X}$ 是欧氏空间、不假设可微性、不假设概率分布、不依赖任何优化算法。这意味着本章揭示的约束——CAC 爆炸、CAB 不可压缩、三元困境、Type B 唯一性——是**拓扑层面的绝对物理极限**，任何具体实现（无论是 Transformer、状态空间模型、还是未来的架构）都无法绕过。系统的表现可以在这些边界之内优化，但边界本身不可突破。

### 开放方向

本章建立了 IDFS 泛化的**骨架理论**——定理链回答了"系统**能**和**不能**做到什么"。以下方向将在后续章节中展开：

- **第二章：拟合形态学**。当 IDFS 确实在逼近目标集 $R$ 时，$F$-空间内部会形成怎样的**内容结构**？不同内容（逻辑规则、知识事实、逐字序列）在 $F$-空间中的几何形态——定义域宽度、Lipschitz 曲率、组合参与度——如何由第一章的定理边界所塑造？这是从"系统的极限"到"系统内部的风景"的转向。
