# 第三章 $\sigma$-$f$ 对偶理论

## 引言：从路由混叠到计算-控制等价

第一章证明了 IDFS $(F, \sigma)$ 在最一般的度量空间假设下的绝对极限，并锁定 Type B（短链高保真）为唯一可行构型。第二章从内容视角刻画了 IDFS 在拟合不同目标时的内部几何形态。本章从**架构视角**追问一个更深层的问题：

> 在什么条件下，路由映射 $\sigma$ 与基函数库 $F$ 可以**精确互相表达**——即 $\sigma$ 的每一次决策都可以被某条 $f$-链复制，而每一条 $f$-链也可以被某个 $\sigma$ 决策编码？

第一章 §2.3 的路由混叠（命题 2.4）已经展示了这种等价的**非受控退化态**：$\sigma(x_2) = \sigma_l(x_1)$ 偶然成立，导致单步拟合与多步复合拟合不可区分。那是一个意外。本章的核心命题是：若对 $\sigma$ 和 $F$ 施加特定的结构约束，路由混叠可以从偶然的退化**升格**为完备的、双向的、精确的**计算-控制等价**——我们称之为 **$\sigma$-$f$ 对偶性**。

这一对偶性不是 IDFS 的固有性质，而是一个**架构级条件**。满足它的 IDFS 获得一种受控的自指结构：系统可以用自己的计算层精确模拟自己的控制层，反之亦然。当此条件与 Type B 的短链约束联合时，IDFS 获得了从"有限函数系统"到"学习与推理的完整框架"的关键跃迁。

---

## §1 $\sigma$-$f$ 对偶的形式定义

### 1.1 $\sigma$-可计算性（$\sigma \to f$）

> **定义**（$\sigma$-可计算性）：设 IDFS $\mathcal{F} = (F, \sigma)$。称 $\sigma$ **$F$-可计算**，若存在有限长 $f$-链 $q_\sigma \in F^*$，使得对所有 $x \in \mathcal{X}$：$\sigma(x) = \text{decode}(q_\sigma(x))$。即路由决策本身可由系统自身的基函数复合精确实现。

### 1.2 $f$-可路由性（$f \to \sigma$）

> **定义**（$f$-可路由性）：称 $F$ 中的任意 $f$-链 $q \in F^*$ **$\sigma$-可路由**，若存在 $x' \in \mathcal{X}$ 使得 $\sigma(x') = q$。即任意计算链路都可由某个路由决策精确访问。

### 1.3 $\sigma$-$f$ 对偶系统

> **定义**（$\sigma$-$f$ 对偶）：称 IDFS $\mathcal{F} = (F, \sigma)$ 满足 $\sigma$-$f$ **完全对偶**，若同时满足 $\sigma$-可计算性与 $f$-可路由性。

## §3 对偶 Type B 系统：学习与推理的统一框架

### 3.1 对偶 Type B 的定义

> **定义**（对偶 Type B 系统）：称 Type B IDFS $\mathcal{F}$ 为**对偶 Type B 系统**，若它同时满足 Type B 约束（$l \le l^*_0$）与 $\sigma$-$f$ 完全对偶。

### 3.2 学习即路由重构

在对偶 Type B 中，"学习"（从训练数据获取拟合能力）等价于构造一个 $\sigma$，使其将输入空间的各区域指派到正确的 $f$-链。由 $\sigma$-$f$ 对偶，此路由构造本身可由 $f$-链实现——**学习过程就是计算过程的一个特例**。

### 3.3 推理即链延伸

"推理"（多步逻辑组合）等价于执行长 $f$-链。在 Type B 中，长链被分段复合替代。由 $\sigma$-$f$ 对偶，段间的 $\sigma$ 重决策等价于执行一段新的 $f$-链——**推理过程中"停下来重新决策"与"继续计算"是同一件事**。

### 3.4 学习与推理的统一

学习 = 路由重构 = $f$-链计算。推理 = 链延伸 = $\sigma$ 重决策。在对偶 Type B 中，二者是 $\sigma$-$f$ 对偶的两个面——**学习和推理在数学上不可区分**。

## §4 对偶性的涌现条件

### 4.1 部分对偶与完全对偶

现实系统中，$\sigma$-$f$ 对偶通常是近似的。定义 $\epsilon$-对偶度，刻画系统距离完全对偶的"缺口"。

### 4.2 对偶相变

猜想：存在临界参数规模 $N^*$，使得 $N < N^*$ 时系统仅具有部分对偶（$\sigma$ 可计算性不完备），$N > N^*$ 时对偶趋于完备。这对应涌现能力的相变点。

## §5 本章总结

$\sigma$-$f$ 对偶理论在 IDFS 框架中引入了一个架构级条件，将 Type B 从一个"受限但可行的工程构型"升格为一个"具有学习与推理统一能力的完整计算框架"。本章的核心逻辑弧线：

$$\text{IDFS} \xrightarrow{\text{Ch01: Type B 唯一性}} \text{Type B} \xrightarrow{\sigma\text{-}f \text{ duality}} \text{对偶 Type B} \xrightarrow{\text{结构后果}} \text{学习/推理统一}$$

对偶性不由 IDFS 定义自动保证——它是一个需要架构设计满足的**额外条件**。第四章及后续章节将考察具体架构（如 Transformer）如何天然实现这一条件。
